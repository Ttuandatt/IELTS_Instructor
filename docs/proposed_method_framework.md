# Vibe Coding Framework v2.0 — Applied to IELTS Helper

- Step 1: Business Idea (Requirements Intake) → Gate 1: Scope Lock.
- Step 2: Low-code (Screens, Flows, Rules, Data) → Gate 2: Design Review.
- Step 3: PRD (01–17 docs) → Gate 3: PRD Approval.
- Step 4: POC (impl + smoke/API tests) → Gate 4: Demo Pass.
- Step 5: Decoupling (refactor, tests) → Gate 5: Code Review.
- Step 6: Delivery (CI/CD, docs, release) → Gate 6: Release Readiness.

Artifacts map to /docs/step1..step4 as scaffolded in this repo. Use Mermaid for diagrams in step3 (14–17).
